# GitHub Push Commands

## Step 1: Create GitHub Repository
1. Go to [GitHub](https://github.com)
2. Click "+" → "New repository"
3. Name: `document-signature-app`
4. Description: "Full-stack document signing application with drag & drop, witness fields, and modern UI"
5. Choose Public/Private
6. **DO NOT** initialize with README, .gitignore, or license
7. Click "Create repository"

## Step 2: Push to GitHub
Run these commands in your terminal:

```bash
cd "d:\web-documnent-signature"
git remote add origin https://github.com/Puneet04-tech/document-signature-app.git
git branch -M main
git push -u origin main
```
